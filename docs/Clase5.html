Clase 5: Modelos lineales

Los modelos lineales son un tipo de modelo estadstico utilizado para predecir el valor de una variable de inters (la variable dependiente) a partir de uno o ms variables predictivas (las variables independientes). Los modelos lineales se basan en la suposicin de que la relacin entre la variable dependiente y las variables independientes es lineal.

Modelo lineal simple

Un modelo lineal simple es un modelo matemtico que se utiliza para predecir una variable dependiente (tambin conocida como la variable objetivo o la variable respuesta) a partir de una nica variable independiente (tambin conocida como la variable predictor o la variable explicativa). Este tipo de modelo se basa en la hiptesis de que la relacin entre la variable dependiente y la variable independiente es lineal, es decir, que puede ser representada por una lnea recta en un grfico.

La forma general de un modelo lineal simple es:

\[y = 0 + 1 * x\]

donde:

  • y es la variable dependiente
  • x es la variable independiente
  • 0 es el intercepto (el valor de y cuando x es igual a cero)
  • 1 es el coeficiente de la variable independiente (la pendiente de la lnea)

Regression lineal

Ahora que hemos encontrado un buen modelo, podemos predicir el valor de abund utilizando la variable saw, para estos es necesario aplicar la regression lineal que se basa en la ecuacion.

\[y = 0 + 1 * x\]

Para aplicar la regresion lineal tenemos que conocer los valores de nuestros elementos, y = es el valor de abund a predecir 0 = 0.099 1 = 0.0058 x = es el valor de saw que utilizaremos para predecir

Si queremos abund cuando tenemos un valor de 75 para saw, seria:

0.099 + 0.0058 * 75
## [1] 0.534

Cuando saw vale 75, abund sera 0.534, estos de acuerdo a nuestro modelo fit2.

Si queremos predecir varios valores a la vez podemos crear una funcion para facilitarnos el trabajo

pre_saw <- function(saw){
  0.099 + 0.0058 * saw
}

probemos nuestra funcion

pre_saw(75)
## [1] 0.534

Ahora con varios valores

pre_saw(1:10)
##  [1] 0.1048 0.1106 0.1164 0.1222 0.1280 0.1338 0.1396 0.1454 0.1512 0.1570

Modelo lineal multiple

Un modelo lineal mltiple es un modelo matemtico que se utiliza para predecir una variable dependiente (tambin conocida como la variable objetivo o la variable respuesta) a partir de varias variables independientes (tambin conocidas como las variables predictoras o las variables explicativas). Este tipo de modelo se basa en la hiptesis de que la relacin entre la variable dependiente y las variables independientes es lineal y que no hay interacciones significativas entre las variables.

La forma general de un modelo lineal mltiple es:

y = 0 + 1 * x1 + 2 * x2 + + n * xn

donde:

  • y es la variable dependiente
  • x1, x2, , xn son las variables independientes
  • 0 es el intercepto (el valor de y cuando todas las variables independientes son iguales a cero)
  • 1, 2, , n son los coeficientes de las variables independientes (la pendiente de la lnea para cada variable independiente)

Modelo lineal con interaccion

Un modelo de regresin lineal con interacciones es un modelo matemtico que se utiliza para predecir una variable dependiente (tambin conocida como la variable objetivo o la variable respuesta) a partir de varias variables independientes (tambin conocidas como las variables predictoras o las variables explicativas), considerando interacciones entre ellas. Una interaccin entre dos variables se refiere a un efecto de una variable sobre la otra que depende del nivel de la otra variable.

La forma general de un modelo de regresin lineal con interacciones es:

y = 0 + 1 * x1 + 2 * x2 + 3 * x1 * x2 + 4 * x3 + + n * xn

donde:

  • y es la variable dependiente
  • x1, x2, x3, , xn son las variables independientes
  • 0 es el intercepto (el valor de y cuando todas las variables independ

Seleccion de modelo usando AIC

La seleccin del modelo ms adecuado para un conjunto de datos es un proceso importante en el anlisis de datos. Existen diferentes criterios que se pueden utilizar para seleccionar el modelo ms adecuado, dependiendo del objetivo del anlisis y del tipo de datos que se tienen. Algunos de los criterios ms comunes son:

  • Simplicidad: Un modelo ms simple es ms fcil de interpretar y generalizar, pero puede no ser tan preciso como un modelo ms complejo.

  • Precisin: Un modelo ms preciso es capaz de explicar mejor la variabilidad de la variable dependiente. Se pueden utilizar mtricas como el coeficiente de determinacin (R cuadrado) o el error cuadrtico medio (MSE) para evaluar la precisin de un modelo.

  • Bondad de ajuste: Un modelo con una bondad de ajuste adecuada se ajusta bien a los datos, pero tambin tiene capacidad de generalizacin a datos futuros. Se pueden utilizar mtricas como el AIC o el BIC para evaluar la bondad de ajuste de un modelo.

  • Interpretabilidad: Un modelo ms fcil de interpretar es ms til para comprender las relaciones entre las variables y para hacer predicciones basadas en ellas.

El AIC (Information Criterion) es una mtrica que se utiliza para evaluar la bondad de ajuste de diferentes modelos de regresin y seleccionar el modelo ms adecuado. El AIC se basa en la idea de que un modelo ms preciso debe tener un menor AIC, pero tambin debe tener una suficiente capacidad de generalizacin a datos futuros.

La frmula del AIC es:

\[AIC = 2 * k - 2 * ln(L)\]

donde:

  • k es el nmero de parmetros del modelo
  • ln(L) es el logaritmo natural del valor de verosimilitud del modelo

A continuacion utilizares el Criterio de Informacion de Akaike (AIC) para identificar el modelo mas adeacuado

Primero debemos de proponer los modelos candidatos

Validacion de modelo

Los modelos lineales se basan en ciertos supuestos acerca de los datos y la relacin entre la variable dependiente y las variables independientes. Estos supuestos son:

Linealidad: La relacin entre la variable dependiente y las variables independientes es lineal. Esto significa que el cambio en la variable dependiente es proporcional al cambio en las variables independientes.

Homoscedasticidad: La varianza de la variable dependiente es constante a lo largo de los diferentes valores de las variables independientes.

Independencia de errores: Los errores (la diferencia entre el valor real y el valor predecido) son independientes entre s.

Normalidad de errores: Los errores siguen una distribucin normal.

Es importante tener en cuenta que si estos supuestos no se cumplen, los resultados del modelo lineal pueden ser inexactos o poco fiables. Por lo tanto, es importante verificar que estos supuestos se cumplen antes de utilizar un modelo lineal.

En R existen diferentes formas de evaluar los supuestos del modelo lineal, por ejemplo podemos utilizar el comando plot().

par(mfrow = c(2,2))

plot(fit3)

colinealidad

La colinealidad es un problema que puede surgir en el anlisis de regresin cuando dos o ms variables independientes estn altamente correlacionadas entre s. Esto puede ocurrir cuando las variables son muy similares o cuando son medidas de manera diferente pero representan el mismo concepto. La colinealidad puede dificultar la interpretacin del modelo y afectar la precisin de las predicciones.

Para detectar la colinealidad en R, puede utilizar la funcin vif() del paquete car. Esta funcin calcula el factor de inflacin de la varianza (VIF) para cada variable del modelo de regresin y permite evaluar la colinealidad entre ellas. El VIF de una variable mide cunto aumenta la varianza de las estimaciones de los coeficientes debido a la colinealidad con otras variables. Valores altos de VIF indican colinealidad y valores bajos indican independencia.
